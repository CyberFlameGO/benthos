Executes a WASI-compliant WASM module.

Introduced in version X.X.X.

```yml
# Config fields, showing default values
label: ""
wasm_wasi:
  path: ""
  io_dir: ""
```

For each message the serialised contents are fed into the WASM module as an input argument. Once the module exits the results written to stdout will replace the message contents.

## Fields

### `path`

The path of the target WASI module to execute.


Type: `string`  

### `io_dir`

An optional directory to store stdout temporary files. When omitted a temporary directory is created.


Type: `string`
